**1. O que é um objeto em programação?**
É algo que guarda dados e ações, representando uma coisa real no código.

**2. O que é uma propriedade de um objeto? Cite um exemplo.**
É uma característica do objeto. Exemplo: `cor = "vermelho"`.

**3. O que é um método? Qual a diferença entre método e propriedade?**
Método é uma ação do objeto; propriedade é um dado dele.

**4. O que é uma classe?**
É o molde usado para criar objetos.

**5. Qual a diferença entre classe e objeto?**
Classe é o modelo; objeto é a coisa criada a partir dele.

**6. Para que serve o constructor dentro de uma classe?**
Para inicializar o objeto quando ele é criado.

**7. O que significa a palavra `this` dentro de uma classe?**
Representa o próprio objeto.

**8. O que a palavra `new` faz ao criar um objeto?**
Cria um novo objeto a partir da classe.

**9. Em TypeScript, por que usamos tipos como string, number e boolean?**
Para evitar erros, garantindo o tipo certo de dado.

**10. Exemplo do mundo real: classe `Aluno`**
- Propriedades: `nome`, `idade`, `nota`
- Método: `aprovar()`
