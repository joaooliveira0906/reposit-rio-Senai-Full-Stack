**1. O que é uma função em programação?**
Uma função é um bloco de código criado para realizar uma tarefa específica.

**2. Qual é a principal vantagem de usar funções em vez de repetir o mesmo código várias vezes?**
Evita repetição de código e facilita a organização e manutenção do programa.

**3. O que são parâmetros de uma função?**
São valores que a função recebe para poder executar uma tarefa.

**4. Qual é a diferença entre parâmetro e argumento?**
Parâmetro é a variável definida na função. Argumento é o valor enviado para ela.

**5. Para que serve a palavra-chave `return`?**
Serve para devolver um valor como resultado da função.

**6. O que acontece quando uma função não possui `return`?**
Ela executa o código, mas não retorna nenhum valor.

**7. O que significa modularizar um código?**
Significa dividir o código em partes menores e organizadas, usando funções ou módulos.

**8. Por que funções pequenas e bem nomeadas facilitam a manutenção do sistema?**
Porque deixam o código mais fácil de entender, corrigir e modificar.

**9. Em TypeScript, por que é útil informar o tipo dos parâmetros e do retorno?**
Porque ajuda a evitar erros e torna o código mais seguro e fácil de compreender.

**10. Observe a função abaixo. Qual será o resultado exibido no console?**
 O resultado exibido no console será **14**, porque a função multiplica o número recebido por 2. Como o valor passado foi **7**, o cálculo é **7 × 2 = 14**.
