**1. O que é uma estrutura condicional em programação? Dê um exemplo do cotidiano usando "se... então... senão".**
É uma forma de o programa tomar decisões.
Exemplo: Se estiver chovendo, então levo guarda-chuva; senão, vou sem.

**2. Qual é a diferença entre usar apenas if e usar if/else?**
O **if** faz algo apenas se a condição for verdadeira.
O **if/else** faz uma coisa se for verdadeira e outra se for falsa.

**3. Quando usamos else if? Crie um exemplo com três ou mais possibilidades.**
Usamos quando existem várias opções.
Exemplo: Se a nota for alta, aprovado; se for média, recuperação; se for baixa, reprovado.

**4. O que significa uma condição retornar true ou false?**
**True** significa verdadeiro.
**False** significa falso.

**5. Qual é a diferença entre == e === em JavaScript? Qual deles é mais recomendado?**
O **==** compara os valores.
O **===** compara os valores e os tipos.
O mais recomendado é o **===**.

**6. Para que servem os operadores &&, || e !? Explique com um exemplo para cada um.**

* **&& (E):** as duas condições precisam ser verdadeiras.
* **|| (OU):** basta uma condição ser verdadeira.
* **! (NÃO):** inverte o resultado da condição.

**7. O que o operador % faz? Por que ele é útil para descobrir se um número é par ou ímpar?**
Ele mostra o resto da divisão.
Se o resto da divisão por 2 for 0, o número é par. Se não for, é ímpar.

**8. O que é switch case? Em que situação ele fica melhor do que vários else if?**
É uma estrutura usada para escolher entre várias opções.
Ela fica melhor quando há muitas opções para comparar.

**9. Para que servem break e default dentro de um switch?**
O **break** encerra a opção escolhida.
O **default** é executado quando nenhuma opção é encontrada.

**10. Por que é importante testar o mesmo programa com vários valores diferentes?**
Para verificar se ele funciona corretamente em diferentes situações e encontrar possíveis erros.
