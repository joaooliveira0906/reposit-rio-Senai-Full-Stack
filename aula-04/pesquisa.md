# Pesquisa - Lacos de Repeticao

## 1. O que e uma estrutura de repeticao?
Resposta:
E um comando da programacao que permite repetir um bloco de codigo varias vezes enquanto uma condicao for verdadeira ou ate um limite definido.

## 2. Para que serve o for?
Resposta:
Serve para repetir um bloco de codigo quando ja sabemos o numero de repeticoes.

## 3. Para que serve o while?
Resposta:
Serve para repetir um bloco de codigo enquanto uma condicao for verdadeira.

## 4. Para que serve o do-while?
Resposta:
Serve para executar um bloco de codigo pelo menos uma vez antes de verificar a condicao.

## 5. Diferenca entre os tres lacos
Resposta:
O for e usado quando sabemos quantas vezes vai repetir.
O while repete enquanto a condicao for verdadeira.
O do-while executa primeiro e depois verifica a condicao.
